# About This Site
Welcome to NAOMINA121! I'm Nao, an information provider and creator. This site shares my blog posts, profile, and activity bases.
## Purpose of the Site
The main purposes of this site are as follows:
- **Information Sharing**: I share my thoughts and experiences through blog posts.
- **Communication**: To foster interaction with visitors and receive feedback and opinions.
- **Providing a Base of Operations**: To introduce my profile and activities, making it easier for interested parties to access me.
## Types of Content
This site provides the following main types of content:
- **Mental Health**: Information and resources concerning mental health.
- **Information Literacy**: Sharing guidelines on accurately evaluating and utilising information.
- **Blog**: Regularly posts articles on various topics.
- **Profile**: Introduces my background, skills, and interests.
- **Activities**: Provides information about my current activities and projects.
